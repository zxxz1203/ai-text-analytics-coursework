# Shapes Dataset Generation

This repository contains the shapes dataset generation pipeline for Task 5.

## Overview
The dataset is generated using a symbolic-first pipeline:

symbolic state -> caption -> canonical label -> rendered image

Each sample contains:
- id
- task
- image_path
- symbolic_state
- caption
- canonical_label
- split

## Vocabulary
- Shapes: circle, square, triangle
- Colors: red, blue, green, yellow
- Sizes: small, large
- Relations: above, below, left of, right of

## Files
- `src/config.py`: fixed vocabulary and settings
- `src/generate_shapes.py`: generates the dataset
- `src/check_shapes.py`: validates metadata and file paths
- `src/summarise_shapes.py`: prints distribution summaries
- `metadata/shapes_metadata.jsonl`: metadata file
- `sample_images/`: example generated images

## Dataset split
- Train: 800
- Validation: 100
- Test: 100

## How to run

Generate the dataset:
```bash
python src/generate_shapes.py
```

Check the dataset:
```bash
python src/check_shapes.py
```
Summarise the dataset:
```bash
python src/summarise_shapes.py
```
